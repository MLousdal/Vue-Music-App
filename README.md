# Vue-Music-App
The product of the "Complete Vue Mastery 2022 (w/ Vuex, Composition API, Router)" course by Zero to Mastery

## What have I learned?

- Vue Router
- Firebase (Authentication, Storage, Collections)
- Tailwind
- Validation (VeeValidate)
- Playing music (Howl)
- Internationalization (i18n)
- Progressive Web App
